# 数据库初始化快速指南

## 📝 步骤说明

### 方法 1: 使用 PowerShell 脚本（推荐）

1. **编辑初始化脚本**
   
   打开 `init_db.ps1`，修改第 7 行的密码：
   ```powershell
   $MYSQL_PASSWORD = "YourPassword"  # 改成你的 MySQL root 密码
   ```

2. **运行脚本**
   ```powershell
   .\init_db.ps1
   ```

3. **更新后端配置**
   
   编辑 `CampusBook\backend\src\main\resources\application.yml`，修改：
   ```yaml
   spring:
     datasource:
       password: YourPassword  # 与上面设置的密码一致
   ```

4. **重启后端**
   ```powershell
   .\run_backend.ps1
   ```

### 方法 2: 使用批处理脚本

1. **编辑批处理文件**
   
   打开 `init_db.bat`，修改第 6 行：
   ```batch
   set MYSQL_PASSWORD=YourPassword
   ```

2. **双击运行** `init_db.bat`

3. **更新后端配置并重启**（同方法 1）

### 方法 3: 手动执行命令

如果知道 MySQL root 密码，可以直接执行：

```powershell
# 1. 创建数据库
mysql -u root -pYourPassword -e "CREATE DATABASE IF NOT EXISTS campusbook DEFAULT CHARACTER SET utf8mb4;"

# 2. 运行初始化脚本（注意使用 Get-Content 管道）
Get-Content "f:\trae 作业\lab2\init_database.sql" | mysql -u root -pYourPassword campusbook

# 3. 验证
mysql -u root -pYourPassword -e "SHOW DATABASES;"
mysql -u root -pYourPassword campusbook -e "SHOW TABLES;"
```

## 🔍 常见问题

### Q: 不知道 MySQL root 密码怎么办？

查看 [`MySQL 配置指南.md`](MySQL 配置指南.md) 中的密码重置步骤。

### Q: 找不到 mysql 命令？

将 MySQL 添加到系统 PATH：
```powershell
$env:Path += ";C:\Program Files\MySQL\MySQL Server 8.0\bin"
```

或者使用完整路径：
```powershell
& "C:\Program Files\MySQL\MySQL Server 8.0\bin\mysql.exe" -u root -pYourPassword ...
```

### Q: 脚本执行报错 "Access denied"？

说明密码不正确，请：
1. 确认 MySQL root 密码
2. 如果忘记密码，需要重置密码（见 MySQL 配置指南）

### Q: 中文乱码？

初始化脚本已设置 utf8mb4 字符集，如仍有问题检查：
- MySQL 配置文件 my.ini 的字符集设置
- 终端编码（PowerShell 默认 UTF-8）
