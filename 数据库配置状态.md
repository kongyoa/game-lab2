### 1. MySQL 密码设置

当前无法使用空密码或默认密码连接 MySQL。

**请按照以下步骤操作：**

#### 选项 A: 如果您知道 MySQL root 密码

1. 编辑文件：`f:\trae 作业\lab2\CampusBook\backend\src\main\resources\application.yml`
2. 修改密码行为实际密码：
   ```yaml
   spring:
     datasource:
       password: YourActualPassword
   ```
3. 运行初始化：
   ```powershell
   mysql -u root -pYourPassword -e "CREATE DATABASE IF NOT EXISTS campusbook DEFAULT CHARACTER SET utf8mb4;"
   mysql -u root -pYourPassword campusbook < "f:\trae 作业\lab2\init_database.sql"
   ```

#### 选项 B: 如果不知道密码

请查看 [`MySQL 配置指南.md`](file://f:/trae 作业/lab2/MySQL 配置指南.md) 获取详细的重置密码步骤。
