## 4. 前端配置建议

### API 基础地址配置
在前端项目中配置 API 代理或基础 URL：

```typescript
// vite.config.ts
export default defineConfig({
  server: {
    proxy: {
      '/api': {
        target: 'http://localhost:8081',
        changeOrigin: true
      }
    }
  }
})
```

## 8. 常见问题

### Q: 端口被占用
**A**: 修改 `application.yml` 中的 `server.port`

### Q: 401 未授权错误
**A**: 需要先登录获取认证 token，或使用 Basic Auth

### Q: 数据库连接失败
**A**: 检查 MySQL 服务是否启动，用户名密码是否正确
